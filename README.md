# Student Marks Calculator

## 📌 Project Description
This is a simple Python project that calculates a student's total marks, percentage, and grade based on marks entered for five subjects.

## ✨ Features
- Takes student name as input
- Accepts marks for 5 subjects
- Calculates total marks
- Calculates percentage
- Displays grade based on percentage
- Prints result in a simple format

## 🛠️ Technologies Used
- Python 3

## 📂 File
- Student marks.py

## ▶️ How to Run
1. Download or clone this repository.
2. Open the project in VS Code or any Python IDE.
3. Run `Student marks.py`.
4. Enter the student's name and marks when prompted.
